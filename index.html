    What is a Turing Machine?

    A Turing Machine (TM) is an abstract mathematical model of computation invented by Alan Turing in 1936. Despite its simplicity, it can compute anything a modern computer can compute — this is the Church-Turing Thesis.

    It consists of:

    • Infinite Tape — divided into cells, each holding one symbol or blank.
    • Read/Write Head — reads, writes, then moves Left (L), Right (R), or Stays (S).
    • State Register — tracks the current state from a finite set Q.
    • Transition Function δ — maps (state, symbol) → (new state, write symbol, direction).

    The machine halts when it enters an accept or reject state.

    📐

    Formal Definition (7-Tuple)

    M = (Q, Σ, Γ, δ, q₀, qacc, qrej)

    QFinite, non-empty set of states
    ΣInput alphabet (blank ∉ Σ)
    ΓTape alphabet (Σ ⊆ Γ, includes blank _)
    δδ: Q × Γ → Q × Γ × {L, R, S}
    q₀Start state (q₀ ∈ Q)
    qaccAccept (halting) state
    qrejReject (halting) state ≠ qacc

    Machine 1 — Palindrome Checker

    Σ = {a, b}

    Accepts strings that read the same forwards and backwards using an erase-and-check strategy:

    1. Read the leftmost un-erased symbol; replace with X; remember it.
    2. Scan all the way to the right end of the tape.
    3. Check if the rightmost un-erased symbol matches. If mismatch → REJECT.
    4. If match → erase it (X) and return to the left. Repeat.
    5. If only X's remain (or tape is empty) → ACCEPT.

    Try: abba ✓ · aba ✓ · abab ✗

    ➕

    Machine 2 — Binary Incrementer

    Σ = {0, 1}

    Adds 1 to a binary number on the tape (MSB left, LSB right):

    1. Scan right to find the end (LSB).
    2. Move left: flip each trailing 1 → 0 (carry propagation).
    3. When a 0 is found → flip to 1 (carry absorbed). ACCEPT.
    4. If the entire number was 1s → overflow: write 1 in the new leading position.

    Try: 101 → 110 · 111 → 1000

    ⚖️

    Machine 3 — aⁿbⁿ Recognizer

    Σ = {a, b}

    Accepts strings of the form aⁿbⁿ (n ≥ 1) — classic context-free language demonstrating TM power:

    1. Cross the leftmost uncrossed a (mark X), scan right.
    2. Cross the first uncrossed b encountered (mark X), return left.
    3. Repeat steps 1–2 until all a's are crossed.
    4. Verify no uncrossed b's remain → ACCEPT. Any remaining → REJECT.
    5. Empty input or b-first string → REJECT.

    Try: ab ✓ · aabb ✓ · aaabbb ✓ · aab ✗

    📋

    Machine 4 — Unary Copy Machine

    Σ = {a}

    Copies a unary string — appends a duplicate after a separator b:

    1. Find the next unmarked a in the original section; replace with A (mark).
    2. Scan right to the end of the tape — if separator b exists, skip copy section; else write it.
    3. Append one new a at the very end of the tape.
    4. Return to leftmost cell and repeat from step 1.
    5. When all original a's are marked (hit b in step 1) → ACCEPT.

    Try: aaa → AAAbaaa · a → Aba
